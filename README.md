# Projeto de Frontend e Backend em React / Frontend and Backend Project in React

Este é um projeto de exemplo que consiste em uma aplicação frontend e backend desenvolvida em React e Node.js, respectivamente. O projeto foi criado para fins de demonstração e aprendizado.

This is a sample project consisting of a frontend and backend application developed in React and Node.js, respectively. The project was created for demonstration and learning purposes.

## Tecnologias Utilizadas / Technologies Used

### Frontend
- React.js
- Bootstrap
- Font Awesome
- React Router
- Axios

### Backend
- Node.js
- JSON Server

This project is divided into two main components: the frontend and the backend. Below are details about each of them:

### Backend

A parte backend do projeto foi desenvolvida em Node.js e usa o JSON Server para simular um servidor RESTful. Alguns detalhes importantes sobre o backend incluem:

The backend part of the project was developed in Node.js and uses JSON Server to simulate a RESTful server. Some important details about the backend include:

- **Arquivo db.json / db.json File :** Um arquivo JSON que atua como um banco de dados simulado. Ele contém uma coleção de usuários de exemplo. / A JSON file that acts as a simulated database. It contains a collection of sample users.

- **Arquivo package.json (Backend) / package.json File :** Este arquivo de configuração define as dependências e scripts para iniciar o servidor JSON Server. / This configuration file defines the dependencies and scripts to start the JSON Server.

## Como Executar o Projeto / How to Run the Project

Para executar o projeto, siga estas etapas: / To run the project, follow these steps:

### Backend (Node.js)

1. Abra o terminal na pasta do backend. / Open the terminal in the backend folder.

2. Execute o comando `npm install` para instalar as dependências. / Run the command npm install to install the dependencies.

3. Execute o comando `npm start` para iniciar o servidor JSON Server. / Run the command npm start to start the JSON Server.

4. O servidor estará disponível em / The server will be available at [http://localhost:3001](http://localhost:3001).

### Frontend (React)

1. Abra o terminal na pasta do frontend. / Open the terminal in the frontend folder.

2. Execute o comando `npm install` para instalar as dependências. / Run the command npm install to install the dependencies

3. Execute o comando `npm start` para iniciar o servidor de desenvolvimento. / Run the command npm start to start the development server.

4. Acesse a aplicação no seu navegador em / Access the application in your browser at : [http://localhost:3000](http://localhost:3000).
